## 형상관리와 Github

### 형상관리의 의미
소프트웨어의 변경사항에 대해 관리하는 것

### 형상관리의 종류

- 통합관리시스템
    - CVS
    - SVN
- 분산관리시스템
    - GIT

### Github
Git 레포지토리를 인터넷 상에서 소프트웨어의 코드를 공유할 수 있도록 호스팅 해주는 서비스

### Github의 장점
- 이슈관리
- 개발진행관리
- 개발코드관리
- 문서관리
- 테스트관리
- 배포관리

즉, 협업 개발을 하기 위해 최적화된 서비스
